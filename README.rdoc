
= DoesFacebook Gem

A paper-thin plugin for parsing Facebook signed request parameters for use in applications
appearing on the Facebook canvas or within Page tabs.


== Installation

To include DoFacebook in your Rails 3 projects, add this dependency to your Gemfile:

gem "dofacebook", ">=0.2", :git=>"git://github.com/awexome/dofacebook.git"
